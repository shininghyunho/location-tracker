## 문서 형식

모든 문서는 두괄식으로 통일한다.

- 머리에 **상태 / 결정 / 대가** 3줄 요약
- 본문은 **배경 → 검토한 대안 → 결과(얻는 것 / 잃는 것·지켜야 할 것)** 순서
